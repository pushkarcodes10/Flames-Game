# FLAMES Game 🔥

A fun and nostalgic FLAMES game that predicts relationship compatibility based on two names using the classic FLAMES algorithm.

FLAMES stands for:
Friends, Lovers, Affection, Marriage, Enemy, Siblings

---

## How It Works

1. Enter two names.
2. Common letters between the names are removed.
3. Remaining letters are counted.
4. The FLAMES algorithm is applied to determine the relationship outcome.

---

## FLAMES Meaning

| Letter | Meaning   |
|--------|-----------|
| F      | Friends   |
| L      | Lovers    |
| A      | Affection |
| M      | Marriage  |
| E      | Enemy     |
| S      | Siblings  |
